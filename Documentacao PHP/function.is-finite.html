is_finite

is_finite — Verifica se um float é um número finito

Descrição

is_finite(float $num): bool

Retorna informando se num é um float finito.

Um float finito não é NAN (is_nan()), ou um infinito (is_infinite()).

Parâmetros

num

O float para verificar

Valor Retornado

true se num não é NAN, INF, -INF, senão retorna false.

Exemplos

Exemplo #1 is_finite() example

<?php
$float = 1.2345;
var_dump($float, is_finite($float));

$nan = sqrt(-1);
var_dump($nan, is_finite($nan));

$inf = 1e308 * 2;
var_dump($inf, is_finite($inf));
?>

O exemplo acima produzirá:

float(1.2345)
bool(true)
float(NAN)
bool(false)
float(INF)
bool(false)
