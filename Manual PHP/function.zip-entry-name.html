zip_entry_name

zip_entry_name — Retorna o nome do arquivo

Aviso

Esta função tornou-se OBSOLETA a partir do PHP 8.0.0. O uso desta função é fortemente desencorajado.

Descrição

zip_entry_name(resource $zip_entry): string|false

Retorna o nome do arquivo especificado.

Parâmetros

zip_entry

A lista de entrada retornada pela zip_read().

Valor Retornado

O nome do arquivo, ou false em caso de falha.

Changelog

Versão Descrição
8.0.0 Esta função está desencorajada em relação a API orientada a objeto, veja ZipArchive::statIndex().
